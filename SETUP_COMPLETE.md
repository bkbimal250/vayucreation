## 🐛 Troubleshooting

### If server won't start:
```bash
# Stop any running Node processes
taskkill /F /IM node.exe

# Restart
npm run dev
```

### If styles don't load:
```bash
# Clear cache and restart
rm -rf node_modules/.vite
npm run dev
```

### If you see dependency errors:
```bash
npm install
npm run dev
```
